Surf the Web

First console to offer online multiplayer and web surfing via a built-in 56K modem. Titles that used online features include Phantasy Start Online and Sonic Adventure.

VMU (Visual Memory Unit)

Only console thus far that used interactive memory units that had a screen. In addition to having clock and calendar features, depending on the title, you could play mini games, trade items in games like Sonic Adventure, even play the main game itself on the small device.
